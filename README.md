## style.go

Console styling library for Golang

![](https://dl.dropboxusercontent.com/s/7ih6ola2zj3ay4r/stylego.png)

## Usage

```go
import . "github.com/azer/style.go"

Style("bold yellow greenBg", "hello!!")
```

Can be also written as:

```js
Style("bold, yellow, greenBg", "hello!!")
```

Or

```js
Style(".bold .yellow .greenBg", "hello!!")
```
